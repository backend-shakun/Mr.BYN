Для корректной работы необходимо:

1) в директории проекта завести sqlite файл 'finance':
    sqlite3 (установить, если не таковой утилиты)
    .open finance (создается файл БД sqllite и открывается консль БД)
2) в директории проекта создать файл credentials.py, где нужно указать token, выданный BotFather для бота:
    token='*************'
3) устанавить зависимости из requirements.txt (лучше делать это в витруальном окружении):
    pip install -r requirements.txt
4) прогнать миграцию БД:
    python3 migrations.py
